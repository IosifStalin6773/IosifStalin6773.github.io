Introducción

FiveM es una de las plataformas más populares para crear servidores personalizados de GTA V. En esta guía completa exploraremos cómo desarrollar mods desde cero, utilizando frameworks como ESX y creando sistemas personalizados que mejorarán la experiencia de juego en tu servidor.

FiveM es una modificación multijugador para GTA V que permite crear servidores personalizados con scripts y recursos personalizados. A diferencia de GTA Online, FiveM da control total sobre el servidor y permite implementar sistemas completamente personalizados.

Características Principales

  • Soporte nativo para Lua y JavaScript
  • API completa para interactuar con el juego
  • Framework de recursos modular
  • Soporte para bases de datos MySQL
  • Comunidad activa y documentación extensiva
  • Compatibilidad con mods personalizados

Arquitectura de Recursos

Cada mod en FiveM se llama "recurso" y tiene una estructura específica:

Estructura de Recurso FiveM
mi-recurso/
    fxmanifest.lua     # Configuración del recurso
    client.lua         # Scripts del cliente
    server.lua         # Scripts del servidor
    shared.lua         # Scripts compartidos
    html/              # Interfaz web (NUI)
    css/               # Estilos para NUI
    js/                # JavaScript para NUI
    images/            # Imágenes y assets

Configuración del Entorno

Antes de comenzar a desarrollar mods para FiveM, necesitas configurar tu entorno de desarrollo correctamente. Esto incluye instalar el software necesario y configurar la estructura básica del servidor.

Software y Extensiones Necesarias

# Extensiones recomendadas para VS Code:

Sistemas Personalizados

Una vez que entiendes los conceptos básicos, puedes crear sistemas personalizados para tu servidor. Aquí mostraremos cómo crear algunos sistemas comunes.

Sistema de Economía

Sistema de Dinero Personalizado
-- server.lua - Sistema de economía
local PlayerMoney = {}

-- Cargar dinero del jugador al conectar
AddEventHandler('playerConnecting', function()
    local source = source
    local identifier = GetPlayerIdentifier(source)
    
    -- Aquí cargarías desde la base de datos
    PlayerMoney[identifier] = 5000 -- Dinero inicial
end)

-- Función para obtener dinero
exports('GetPlayerMoney', function(source)
    local identifier = GetPlayerIdentifier(source)
    return PlayerMoney[identifier] or 0
end)

-- Función para añadir dinero
exports('AddPlayerMoney', function(source, amount)
    local identifier = GetPlayerIdentifier(source)
    PlayerMoney[identifier] = (PlayerMoney[identifier] or 0) + amount
    
    -- Notificar al cliente
    TriggerClientEvent('money:updated', source, PlayerMoney[identifier])
end)

-- Función para remover dinero
exports('RemovePlayerMoney', function(source, amount)
    local identifier = GetPlayerIdentifier(source)
    local currentMoney = PlayerMoney[identifier] or 0
    
    if currentMoney >= amount then
        PlayerMoney[identifier] = currentMoney - amount
        TriggerClientEvent('money:updated', source, PlayerMoney[identifier])
        return true
    else
        return false
    end
end)

Sistema de Inventario

Inventario Básico
-- shared.lua - Sistema de inventario
local Inventory = {}

-- Estructura de item
local ItemTypes = {
    ['phone'] = {name = 'Teléfono', weight = 0.5, stackable = false},
    ['water'] = {name = 'Agua', weight = 0.3, stackable = true},
    ['bread'] = {name = 'Pan', weight = 0.2, stackable = true},
    ['lockpick'] = {name = 'Ganzúa', weight = 0.1, stackable = true}
}

-- Función para crear inventario
function CreateInventory(identifier)
    Inventory[identifier] = {
        items = {},
        maxWeight = 30.0,
        currentWeight = 0.0
    }
    return Inventory[identifier]
end

-- Función para añadir item
function AddItem(identifier, itemName, count)
    local inventory = Inventory[identifier]
    if not inventory then
        inventory = CreateInventory(identifier)
    end
    
    local itemType = ItemTypes[itemName]
    if not itemType then return false end
    
    local totalWeight = itemType.weight * count
    if inventory.currentWeight + totalWeight > inventory.maxWeight then
        return false
    end
    
    if itemType.stackable then
        inventory.items[itemName] = (inventory.items[itemName] or 0) + count
    else
        for i = 1, count do
            table.insert(inventory.items, {
                name = itemName,
                label = itemType.name,
                weight = itemType.weight
            })
        end
    end
    
    inventory.currentWeight = inventory.currentWeight + totalWeight
    return true
end

JavaScript para Interactividad

script.js - Lógica del Menú
// script.js
document.addEventListener('DOMContentLoaded', function() {
    const menuContainer = document.getElementById('menuContainer');
    const closeBtn = document.getElementById('closeBtn');
    const menuItems = document.querySelectorAll('.menu-item');
    
    // Cerrar menú
    closeBtn.addEventListener('click', function() {
        fetch(`https://${GetParentResourceName()}/closeMenu`, {
            method: 'POST',
            headers: {
                'Content-Type': 'application/json; charset=UTF-8',
            },
            body: JSON.stringify({})
        });
    });
    
    // Manejar clics en items del menú
    menuItems.forEach(item => {
        item.addEventListener('click', function() {
            const action = this.getAttribute('data-action');
            
            fetch(`https://${GetParentResourceName()}/menuAction`, {
                method: 'POST',
                headers: {
                    'Content-Type': 'application/json; charset=UTF-8',
                },
                body: JSON.stringify({
                    action: action
                })
            });
        });
    });
    
    // Escuchar mensajes del cliente
    window.addEventListener('message', function(event) {
        const data = event.data;
        
        switch (data.type) {
            case 'showMenu':
                menuContainer.style.display = 'block';
                break;
            case 'hideMenu':
                menuContainer.style.display = 'none';
                break;
            case 'updateMoney':
                // Actualizar display de dinero
                break;
        }
    });
});

// Función para comunicarse con el cliente
function sendMessage(type, data) {
    fetch(`https://${GetParentResourceName()}/message`, {
        method: 'POST',
        headers: {
            'Content-Type': 'application/json; charset=UTF-8',
        },
        body: JSON.stringify({
            type: type,
            data: data
        })
    });
}

Integración con Lua

client.lua - Control de NUI
-- client.lua - Control de NUI
local isMenuOpen = false

-- Abrir menú
function OpenMenu()
    if not isMenuOpen then
        isMenuOpen = true
        SetNuiFocus(true, true)
        SendNUIMessage({
            type = 'showMenu'
        })
    end
end

-- Cerrar menú
function CloseMenu()
    if isMenuOpen then
        isMenuOpen = false
        SetNuiFocus(false, false)
        SendNUIMessage({
            type = 'hideMenu'
        })
    end
end

-- Callbacks de NUI
RegisterNUICallback('closeMenu', function(data, cb)
    CloseMenu()
    cb('ok')
end)

RegisterNUICallback('menuAction', function(data, cb)
    local action = data.action
    
    -- Procesar acción del menú
    if action == 'inventory' then
        -- Abrir inventario
        TriggerEvent('inventory:open')
    elseif action == 'wallet' then
        -- Mostrar dinero
        TriggerEvent('wallet:show')
    elseif action == 'phone' then
        -- Abrir teléfono
        TriggerEvent('phone:open')
    elseif action == 'garage' then
        -- Abrir garaje
        TriggerEvent('garage:open')
    elseif action == 'gps' then
        -- Abrir GPS
        TriggerEvent('gps:open')
    elseif action == 'settings' then
        -- Abrir configuración
        TriggerEvent('settings:open')
    elseif action == 'help' then
        -- Mostrar ayuda
        TriggerEvent('help:show')
    end
    
    cb('ok')
end)

-- Comando para abrir menú
RegisterCommand('menu', function()
    OpenMenu()
end, false)

-- Tecla para abrir/cerrar menú
Citizen.CreateThread(function()

Conexión a Base de Datos

Lua - Conexión MySQL
-- server.lua - Conexión MySQL
local MySQL = exports['oxmysql']

-- Función para cargar datos del jugador
function LoadPlayerData(identifier)
    MySQL.scalar('SELECT money, bank, job FROM players WHERE identifier = ?', {
        identifier
    }, function(result)
        if result then
            -- Enviar datos al cliente
            TriggerClientEvent('playerData:loaded', source, {
                money = result.money,
                bank = result.bank,
                job = result.job
            })
        else
            -- Crear nuevo jugador
            MySQL.execute('INSERT INTO players (identifier, name, money, bank, job) VALUES (?, ?, ?, ?, ?)', {
                identifier,
                GetPlayerName(source),
                5000,
                10000,
                'unemployed'
            })
            
            TriggerClientEvent('playerData:loaded', source, {
                money = 5000,
                bank = 10000,
                job = 'unemployed'
            })
        end
    end)
end

-- Guardar datos del jugador
function SavePlayerData(identifier, data)
    MySQL.execute('UPDATE players SET money = ?, bank = ?, job = ? WHERE identifier = ?', {
        data.money,
        data.bank,
        data.job,
        identifier
    })
end

-- Evento cuando el jugador se conecta
AddEventHandler('playerConnecting', function()
    local source = source
    local identifier = GetPlayerIdentifier(source)
    
    -- Cargar datos después de que el jugador se una completamente
    Citizen.CreateThread(function()
        Citizen.Wait(2000)
        LoadPlayerData(identifier)
    end)
end)

-- Guardar datos cuando el jugador se desconecta
AddEventHandler('playerDropped', function(reason)
    local source = source
    local identifier = GetPlayerIdentifier(source)
    
    -- Obtener datos actuales del jugador
    TriggerClientEvent('playerData:getData', source)
end)

-- Recibir datos del cliente para guardar
RegisterNetEvent('playerData:saveData')
AddEventHandler('playerData:saveData', function(data)
    local source = source
    local identifier = GetPlayerIdentifier(source)
    
    SavePlayerData(identifier, data)
end)

Mejores Prácticas

Para crear mods de alta calidad y mantenibles, es importante seguir mejores prácticas de desarrollo y optimización.

Optimización de Rendimiento

  • Usa Citizen.CreateThread con delays apropiados
  • Evita loops infinitos sin Citizen.Wait
  • Cachea referencias a funciones y variables
  • Usa eventos en lugar de polling constante
  • Optimiza consultas a base de datos

Código Limpio

  • Usa nombres descriptivos para variables y funciones
  • Comenta código complejo
  • Separa la lógica en módulos
  • Usa configuración externa cuando sea posible
  • Sigue convenciones de nomenclatura consistentes

Seguridad

  • Valida todas las entradas del cliente
  • Usa permisos para comandos administrativos
  • Sanitiza datos de base de datos
  • Implementa rate limiting para acciones
  • Usa HTTPS para recursos web

-- Obtener inventario del jugador
ESX.RegisterServerCallback('inventory:getPlayerInventory', function(source, cb)
    local xPlayer = ESX.GetPlayerFromId(source)
    local inventory = xPlayer.getInventory()
    
    local items = {}
    local totalWeight = 0.0
    
    for k, v in pairs(inventory) do
        if v.count > 0 then
            local item = Config.Inventory.items[v.name]
            if item then
                totalWeight = totalWeight + (item.weight * v.count)
                
                table.insert(items, {
                    name = v.name,
                    label = item.label,
                    count = v.count,
                    weight = item.weight,
                    description = item.description,
                    stackable = item.stackable
                })
            end
        end
    end
    
    cb({
        items = items,
        totalWeight = totalWeight,
        maxWeight = Config.Inventory.maxWeight,
        maxSlots = Config.Inventory.maxSlots
    })
end)

-- Añadir item al inventario
RegisterServerEvent('inventory:addItem')
AddEventHandler('inventory:addItem', function(source, itemName, count)
    local xPlayer = ESX.GetPlayerFromId(source)
    local item = Config.Inventory.items[itemName]
    
    if not item then
        TriggerClientEvent('inventory:notification', source, 'error', 'Item no válido')
        return
    end
    
    -- Verificar peso máximo
    ESX.TriggerServerCallback('inventory:getPlayerInventory', source, function(inventory)
        local newWeight = inventory.totalWeight + (item.weight * count)
        
        if newWeight > inventory.maxWeight then
            TriggerClientEvent('inventory:notification', source, 'error', 'Inventario lleno')
            return
        end
        
        -- Verificar slots disponibles
        if not item.stackable and #inventory.items >= inventory.maxSlots then
            TriggerClientEvent('inventory:notification', source, 'error', 'Sin slots disponibles')
            return
        end
        
        -- Añadir item
        xPlayer.addInventoryItem(itemName, count)
        TriggerClientEvent('inventory:notification', source, 'success', 'Item añadido')
        TriggerClientEvent('inventory:updateInventory', source)
    end)
end)

function OpenInventory()
    if isInventoryOpen then
        return
    end
    
    isInventoryOpen = true
    
    -- Obtener datos del inventario
    ESX.TriggerServerCallback('inventory:getPlayerInventory', function(inventory)
        currentInventory = inventory
        RenderInventory()
    end)
end

function RenderInventory()
    local elements = {}
    
    -- Información del inventario
    table.insert(elements, {
        label = 'Peso: ' .. string.format("%.1f", currentInventory.totalWeight) .. '/' .. currentInventory.maxWeight .. ' kg',
        value = 'info'
    })
    
    table.insert(elements, {
        label = 'Slots: ' .. #currentInventory.items .. '/' .. currentInventory.maxSlots,
        value = 'info'
    })
    
    table.insert(elements, {label = '---', value = 'separator'})
    
    -- Items del inventario
    for _, item in ipairs(currentInventory.items) do
        local label = item.label
        
        if item.count > 1 then
            label = label .. ' x' .. item.count
        end
        
        label = label .. ' (' .. string.format("%.1f", item.weight * item.count) .. ' kg)'
        
        table.insert(elements, {
            label = label,
            value = item.name,
            item = item
        })
    end
    
    ESX.UI.Menu.Open('default', GetCurrentResourceName(), 'inventory_menu', {
        title = 'Inventario',
        align = 'bottom-right',
        elements = elements
    }, function(data, menu)
        local action = data.current.value
        
        if action == 'info' then
            -- Mostrar información
            return
        end
        
        -- Menú de acciones para el item
        OpenItemActions(data.current.item)
    end, function(data, menu)
        menu.close()
        isInventoryOpen = false
    end)
end

function OpenItemActions(item)
    local elements = {}
    
    -- Usar item
    table.insert(elements, {
        label = 'Usar',
        value = 'use'
    })
    
    -- Soltar item
    table.insert(elements, {
        label = 'Soltar',
        value = 'drop'
    })
    
    -- Si es stackable, permitir soltar cantidad
    if item.stackable and item.count > 1 then
        table.insert(elements, {
            label = 'Soltar cantidad',
            value = 'drop_amount'
        })
    end
    
    ESX.UI.Menu.Open('default', GetCurrentResourceName(), 'item_actions', {
        title = item.label,
        align = 'bottom-right',
        elements = elements
    }, function(data, menu)
        local action = data.current.value
        
        if action == 'use' then
            TriggerServerEvent('inventory:useItem', item.name)
        elseif action == 'drop' then
            TriggerServerEvent('inventory:dropItem', item.name, 1)
        elseif action == 'drop_amount' then
            -- Diálogo para cantidad
            ESX.UI.Menu.Open('dialog', GetCurrentResourceName(), 'drop_amount_dialog', {
                title = 'Cantidad a soltar'
            }, function(data2, menu2)
                local amount = tonumber(data2.value)
                
                if amount and amount > 0 and amount <= item.count then
                    TriggerServerEvent('inventory:dropItem', item.name, amount)
                    menu2.close()
                    menu.close()
                else
                    ESX.ShowNotification('Cantidad inválida')
                end
            end, function(data2, menu2)
                menu2.close()
            end)
        end
    end, function(data, menu)
        menu.close()
    end)
end

JavaScript para Interfaz

// js/app.js
class FiveMInterface {
    constructor() {
        this.isVisible = false;
        this.currentTab = 'main';
        this.playerData = {};
        
        this.init();
    }
    
    init() {
        // Escuchar mensajes del juego
        window.addEventListener('message', (event) => {
            this.handleMessage(event.data);
        });
        
        // Configurar event listeners
        this.setupEventListeners();
        
        // Ocultar interfaz inicialmente
        this.hide();
    }
    
    setupEventListeners() {
        // Botón cerrar
        document.getElementById('closeBtn').addEventListener('click', () => {
            this.hide();
            this.sendMessage('closeInterface');
        });
        
        // Tabs
        document.querySelectorAll('.tab-btn').forEach(btn => {
            btn.addEventListener('click', (e) => {
                this.switchTab(e.target.dataset.tab);
            });
        });
        
        // Acciones
        document.getElementById('withdrawBtn').addEventListener('click', () => {
            this.openAmountDialog('withdraw');
        });
        
        document.getElementById('depositBtn').addEventListener('click', () => {
            this.openAmountDialog('deposit');
        });
        
        // Tecla ESC para cerrar
        document.addEventListener('keydown', (e) => {
            if (e.key === 'Escape' && this.isVisible) {
                this.hide();
                this.sendMessage('closeInterface');
            }
        });
    }
    
    handleMessage(data) {
        switch (data.action) {
            case 'showInterface':
                this.show();
                if (data.playerData) {
                    this.updatePlayerData(data.playerData);
                }
                break;
                
            case 'hideInterface':
                this.hide();
                break;
                
            case 'updatePlayerData':
                this.updatePlayerData(data.playerData);
                break;
                
            case 'updateInventory':
                this.updateInventory(data.inventory);
                break;
                
            case 'showNotification':
                this.showNotification(data.type, data.message);
                break;
        }
    }
    
    show() {
        document.getElementById('app').classList.remove('hidden');
        this.isVisible = true;
        this.sendMessage('interfaceShown');
    }
    
    hide() {
        document.getElementById('app').classList.add('hidden');
        this.isVisible = false;
    }
    
    switchTab(tabName) {
        // Actualizar botones
        document.querySelectorAll('.tab-btn').forEach(btn => {
            btn.classList.remove('active');
        });
        document.querySelector(`[data-tab="${tabName}"]`).classList.add('active');
        
        // Actualizar contenido
        document.querySelectorAll('.tab-pane').forEach(pane => {
            pane.classList.remove('active');
        });
        document.getElementById(tabName).classList.add('active');
        
        this.currentTab = tabName;
    }
    
    updatePlayerData(data) {
        this.playerData = data;
        
        // Actualizar estadísticas principales
        document.getElementById('money').textContent = this.formatMoney(data.money || 0);
        document.getElementById('bank').textContent = this.formatMoney(data.bank || 0);
        document.getElementById('items').textContent = data.inventory ? data.inventory.length : 0;
        
        // Actualizar estadísticas detalladas
        document.getElementById('playTime').textContent = this.formatTime(data.playTime || 0);
        document.getElementById('missions').textContent = data.missions || 0;
        document.getElementById('level').textContent = data.level || 1;
        
        // Actualizar inventario
        if (data.inventory) {
            this.updateInventory(data.inventory);
        }
    }
    
    updateInventory(inventory) {
        const grid = document.getElementById('inventoryGrid');
        grid.innerHTML = '';
        
        inventory.forEach(item => {
            const itemElement = this.createInventoryItem(item);
            grid.appendChild(itemElement);
        });
    }
    
    createInventoryItem(item) {
        const div = document.createElement('div');
        div.className = 'inventory-item';
        
        const icon = this.getItemIcon(item.name);
        const label = item.label || item.name;
        const count = item.count || 1;
        
        div.innerHTML = `
            
            
${label}

            
x${count}

        `;
        
        div.addEventListener('click', () => {
            this.selectItem(item);
        });
        
        return div;
    }
    
    getItemIcon(itemName) {
        const icons = {
            'bread': 'fas fa-bread-slice',
            'water': 'fas fa-tint',
            'phone': 'fas fa-mobile-alt',
            'money': 'fas fa-money-bill',
            'weapon': 'fas fa-gun'
        };
        
        return icons[itemName] || 'fas fa-box';
    }
    
    selectItem(item) {
        this.sendMessage('itemSelected', {
            name: item.name,
            label: item.label,
            count: item.count
        });
    }
    
    openAmountDialog(type) {
        const amount = prompt(`Cantidad para ${type === 'withdraw' ? 'retirar' : 'depositar'}:`);
        
        if (amount && !isNaN(amount) && amount > 0) {
            this.sendMessage(type, {
                amount: parseInt(amount)
            });
        }
    }
    
    showNotification(type, message) {
        // Crear elemento de notificación
        const notification = document.createElement('div');
        notification.className = `notification ${type}`;
        notification.textContent = message;
        
        // Estilos
        Object.assign(notification.style, {
            position: 'fixed',
            top: '20px',
            right: '20px',
            padding: '15px 20px',
            borderRadius: '8px',
            color: '#fff',
            fontWeight: '500',
            zIndex: '10000',
            transform: 'translateX(100%)',
            transition: 'transform 0.3s ease'
        });
        
        // Color según tipo
        const colors = {
            success: '#4CAF50',
            error: '#f44336',
            info: '#2196F3',
            warning: '#ff9800'
        };
        
        notification.style.background = colors[type] || colors.info;
        
        document.body.appendChild(notification);
        
        // Animar entrada
        setTimeout(() => {
            notification.style.transform = 'translateX(0)';
        }, 100);
        
        // Remover después de 3 segundos
        setTimeout(() => {
            notification.style.transform = 'translateX(100%)';
            setTimeout(() => {
                document.body.removeChild(notification);
            }, 300);
        }, 3000);
    }
    
    formatMoney(amount) {
        return new Intl.NumberFormat('es-ES', {
            style: 'currency',
            currency: 'USD',
            minimumFractionDigits: 0
        }).format(amount);
    }
    
    formatTime(minutes) {
        const hours = Math.floor(minutes / 60);
        const mins = minutes % 60;
        
        if (hours > 0) {
            return `${hours}h ${mins}m`;
        }
        
        return `${mins}m`;
    }
    
    sendMessage(action, data = {}) {
        fetch(`https://${GetParentResourceName()}/${action}`, {
            method: 'POST',
            headers: {
                'Content-Type': 'application/json; charset=UTF-8',
            },
            body: JSON.stringify(data)
        });
    }
}

Mejores Prácticas

Optimización de Rendimiento

  • Usar Citizen.Wait() en bucles para no bloquear el juego
  • Limitar actualizaciones de NUI a lo necesario
  • Cache de datos frecuentemente accedidos
  • Usar callbacks en lugar de polling constante

Seguridad

  • Validar todas las entradas del cliente en el servidor
  • Usar permisos y roles para controlar acceso
  • No confiar en datos del cliente para operaciones críticas
  • Implementar rate limiting para comandos

Código Limpio

  • Separar lógica de cliente y servidor
  • Usar nombres descriptivos para variables y funciones
  • Comentar código complejo
  • Usar configuración externa para valores ajustables

Testing

  • Probar recursos con múltiples jugadores
  • Verificar rendimiento bajo carga
  • Testear edge cases y errores
  • Documentar instalación y configuración

Conclusiones

El desarrollo de mods para FiveM es un campo fascinante que combina creatividad con habilidades técnicas. Con esta guía tienes las bases para crear sistemas complejos y mejorar la experiencia de juego en tu servidor.

Puntos Clave:

  • Comienza con recursos simples y ve aumentando complejidad
  • Usa frameworks establecidos como ESX o QBCore
  • Optimiza siempre el rendimiento y la seguridad
  • Documenta tu código para facilitar mantenimiento
  • Participa en la comunidad para aprender y compartir
