Escape From Tarkov Helix Framework

Sistema de Inventario Avanzado

Sistema de inventario inspirado en EFT implementado en el framework Helix para Garry's Mod. Recrea la mecánica de gestión de objetos e inventario de EFT con cuadrícula avanzada, interfaz visual intuitiva y compatibilidad total con el ecosistema Helix.

Ver en GitHub Volver a Proyectos

Detalles del Proyecto

Características Principales

  • Sistema de inventario con cuadrícula al estilo EFT
  • Gestión avanzada de objetos y pesos
  • Interfaz visual intuitiva inspirada en Escape From Tarkov
  • Integración completa con Helix Framework
  • Soporte para objetos personalizables
  • Compatibilidad con armas, equipamiento y consumibles

Tecnologías Utilizadas

Lua Scripting
Helix Framework
Garry's Mod
UI Design
MySQL
Networking

Sistema de Inventario

  • Cuadrícula de inventario configurable (6x8, 8x10, etc.)
  • Sistema de peso y espacio realista
  • Apilamiento automático de objetos compatibles
  • Categorización inteligente de items
  • Búsqueda y filtrado rápido de objetos
  • Drag & drop intuitivo para gestión

Características de Seguridad

  • Validación de objetos en servidor
  • Protección contra exploits de inventario
  • Sistema de logs completo
  • Integración con sistema de permisos Helix
  • Anti-cheat integrado
  • Rollback automático en caso de errores

Instalación y Configuración

1

Requisitos Previos

- Garry's Mod Server
- Helix Framework instalado
- Acceso administrativo al servidor
- MySQL para base de datos
2

Instalación del Mod

1. Clonar el repositorio
2. Copiar archivos a garrysmod/addons
3. Reiniciar el servidor
4. Configurar en panel de admin
3

Configuración Base

-- Configurar tamaño de inventario
-- Definir objetos personalizados
-- Ajustar pesos y valores
-- Establecer permisos de acceso
4

Personalización

-- Modificar UI según preferencias
-- Añadir objetos personalizados
-- Configurar reglas del servidor
-- Integrar con otros mods
