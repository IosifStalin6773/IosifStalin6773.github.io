ALL-IN-ONE-BOT

Bot Multifuncional para Discord

Un bot Discord completo con capacidades de moderación, entretenimiento y automatización. Fork mejorado del proyecto original con características personalizadas y optimizaciones de rendimiento.

Detalles del Proyecto

Características Principales

  • Sistema de moderación avanzado con bans, kicks y mute
  • Comandos de entretenimiento y juegos interactivos
  • Automatización de tareas y bienvenidas personalizadas
  • Sistema de roles y permisos completamente personalizable
  • Estadísticas y analíticas del servidor

Soporte

Instalación y Configuración

1

Clonar el Repositorio

git clone https://github.com/IosifStalin6773/ALL-IN-ONE-BOT.git
cd ALL-IN-ONE-BOT
2

Instalar Dependencias

npm install
3

Configurar Variables de Entorno

cp .env.example .env
# Editar .env con tu token de Discord y otras configuraciones
4

Iniciar el Bot

npm start
